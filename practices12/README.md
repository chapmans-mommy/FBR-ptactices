В рамках выполнения практических заданий 7-11 было разработано веб-приложение на Node.js (Express) + React, реализующее интернет-магазин с системой аутентификации, JWT-токенами и ролевой моделью доступа (RBAC).

1 Практическое занятие №7 — Хеширование паролей (bcrypt)
Реализована регистрация и вход с хешированием паролей через bcrypt

2 Практическое занятие №8 — JWT токены
Добавлена генерация access-токена при входе
Создан middleware authMiddleware для защиты маршрутов
Добавлен защищённый маршрут GET /me

3 Практическое занятие №9 — Refresh-токены
Введены refresh-токены для обновления access-токенов
Реализован маршрут POST /refresh для получения новой пары токенов
Настроена ротация refresh-токенов

4 Практическое занятие №10 — Хранение токенов на фронтенде
React-приложение с хранением токенов в localStorage
Добавлены перехватчики axios для автоматической подстановки токена и обработки 401 ошибок
Реализован таймер обратного отсчёта времени жизни токена (15 секунд для демонстрации)

5 Практическое занятие №11 — Управление доступом (RBAC)
В модель пользователя добавлено поле role (user, seller, admin)
Создан middleware roleMiddleware для проверки прав доступа
Реализовано управление пользователями (только для admin)
Настроено отображение кнопок и форм на фронтенде в зависимости от роли

📋 Список маршрутов API
🔹 Публичные маршруты (доступны без токена)
Метод	Маршрут	Описание
POST	/api/auth/register	Регистрация нового пользователя
POST	/api/auth/login	Вход в систему
POST	/api/auth/refresh	Обновление пары токенов
🔹 Защищённые маршруты (требуют access-токен)
Метод	Маршрут	Доступ	Описание
GET	/api/auth/me	Любой авторизованный	Получение данных текущего пользователя
GET	/api/products	Любой авторизованный	Список товаров
GET	/api/products/:id	Любой авторизованный	Получение товара по ID
POST	/api/products	Продавец, Админ	Создание товара
PUT	/api/products/:id	Продавец, Админ	Обновление товара
DELETE	/api/products/:id	Админ	Удаление товара
GET	/api/users	Админ	Список пользователей
GET	/api/users/:id	Админ	Получение пользователя по ID
PUT	/api/users/:id	Админ	Обновление данных пользователя
DELETE	/api/users/:id	Админ	Удаление пользователя
