# 📘 Отчёт по практическим занятиям №13–17  
### *Frontend & Backend разработка*  
**Тема:** Прогрессивные веб-приложения (PWA), Service Worker, Push-уведомления, напоминания

---

## Обзор

В ходе выполнения практических работ №13–17 было разработано **офлайн-приложение для заметок**, которое:

-  Работает без интернета
-  Устанавливается на устройство как PWA
-  Отправляет push-уведомления
-  Позволяет планировать напоминания и откладывать их

---

## Практическая №13 — Service Worker (офлайн)

> **Цель:** Научить приложение работать без интернета.

### Что сделано:
- Зарегистрирован **Service Worker**
- Закэшированы статические файлы (HTML, CSS, JS)
- Реализована стратегия **Cache First** для статики
- Приложение полностью работает **офлайн**

### Кэшированные файлы:
| Файл | Тип |
|------|-----|
| index.html | App Shell |
| style.css | Стили |
| app.js | Логика |
| manifest.json | PWA |
| иконки | PNG |

---

## Практическая №14 — Web App Manifest (PWA)

> **Цель:** Добавить возможность установки приложения на устройство.

###  Что сделано:
- Создан файл `manifest.json`
- Добавлены иконки разных размеров (16×16 … 512×512)
- Подключены мета-теги для iOS и Android
- Приложение можно **установить** как нативное


---

## Практическая №15 — HTTPS + App Shell

> **Цель:** Обеспечить безопасное соединение и мгновенную загрузку интерфейса.

### Что сделано:
- Настроен **локальный HTTPS** через `mkcert`
- Реализована архитектура **App Shell**
- Статика (каркас) кэшируется сразу
- Контент подгружается динамически


---

## Практическая №16 — WebSocket + Push

> **Цель:** Добавить реальное время и push-уведомления.

### Что сделано:
- Реализован **WebSocket-сервер** (`socket.io`)
- При добавлении заметки событие рассылается всем клиентам
- Добавлены **push-уведомления** через `web-push`
- Пользователь может включить/отключить уведомления


---

## Практическая №17 — Детализация Push (напоминания)

> **Цель:** Реализовать отложенные напоминания и кнопку "Отложить".

### Что сделано:
- Добавлена форма с выбором **даты и времени**
- На сервере — `setTimeout` для планирования
- Push-уведомление приходит в заданное время
- В уведомлении — кнопка **"Отложить на 5 минут"**
- При нажатии сервер переносит таймер
